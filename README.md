# 🚀 Crypto Volatility Dashboard

A machine learning-powered web application that analyzes historical cryptocurrency market data to predict price volatility and provide interactive visual insights. The dashboard combines predictive analytics with a modern user interface to help users understand market trends and make data-driven decisions.

---

## 📌 Features

* 📈 Predict cryptocurrency price volatility using machine learning models.
* 📊 Interactive dashboard for visualizing market trends.
* 🔍 Historical cryptocurrency data analysis.
* ⚡ Fast and responsive user interface built with modern web technologies.
* 📉 Data preprocessing and feature engineering for improved model performance.
* 🔄 Scalable architecture with frontend and backend integration.

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/Mayur07-ops/crypto-volatility-dashboard.git
```

### Navigate to the project

```bash
cd crypto-volatility-dashboard
```

### Install dependencies

```bash
npm install
```

or

```bash
pnpm install
```

### Start the development server

```bash
npm run dev
```

or

```bash
pnpm dev
```
